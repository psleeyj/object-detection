# Real-Time Object Detection

A computer vision project that performs real-time object detection using YOLOv8 and webcam input.

## Features
- Real-time webcam object detection
- Object labeling with confidence scores
- Bounding box visualization
- YOLOv8 pretrained model

## Technologies
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## Installation

```bash
pip install ultralytics opencv-python
```

## Run

```bash
python detect.py
```
